# Lectura Viva Rural - Versión HTML/GitHub Pages corregida

Corrección principal:
- Los 20 recuadros de actividades ahora tienen una lista interna de respaldo en `js/app.js`.
- Si el navegador bloquea `data/actividades.json`, las actividades se cargan igual.
- Puede abrirse directamente con doble clic en `index.html` o publicarse en GitHub Pages.

Publicación:
1. Crear repositorio en GitHub.
2. Subir todos los archivos y carpetas.
3. Ir a Settings > Pages.
4. Seleccionar rama main y carpeta root.
5. Abrir la URL generada.

Limitación:
- GitHub Pages no guarda archivos en el servidor. Las evidencias quedan en el navegador.
